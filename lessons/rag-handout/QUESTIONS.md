# 예시 질의응답 시나리오


## 1. 정확한 키워드가 들어간 질문

- 감정 분석 웹앱은 어떤 기술을 사용했나요?
- Firebase를 사용한 프로젝트는 무엇인가요?
- ChromaDB가 들어간 프로젝트를 알려주세요.
- Raspberry Pi를 사용한 프로젝트의 성과는 무엇인가요?

## 2. 표현은 다르지만 의미가 비슷한 질문

- 팀으로 협업한 경험이 잘 드러나는 프로젝트는?
- 데이터를 모아서 분석한 프로젝트를 추천해줘.
- 사용자의 입력을 받아 자동으로 피드백하거나 추천한 프로젝트는?
- 학교 생활의 문제를 해결하려고 만든 서비스는?

## 3. 여러 프로젝트가 함께 검색될 수 있는 비교 질문

- FastAPI를 쓴 프로젝트들을 비교해서 설명해줘.
- Python을 사용한 프로젝트 중 가장 성과가 뚜렷한 것은?
- 챗봇 또는 생성형 AI와 관련된 프로젝트는 어떤 차이가 있어?
- 대시보드가 포함된 프로젝트들을 찾아서 역할을 비교해줘.

## 4. 역할과 성과를 묻는 면접형 질문

- 본인이 직접 맡은 역할이 가장 큰 프로젝트는 무엇인가요?
- 수치로 성과를 설명할 수 있는 프로젝트를 알려주세요.
- 리더십 경험이 드러나는 프로젝트는 무엇인가요?
- 백엔드 API를 만든 경험을 설명해 주세요.

## 5. 답변 불가 확인 질문

- 지원자의 희망 연봉은 얼마인가요?
- 대학 전공은 무엇인가요?
- 회사 인턴 경험이 있나요?
- 영어 성적은 몇 점인가요?

## 6. 검색 실험 과제

1. `04_search_test.py`에서 `k=1`, `k=2`, `k=4`로 바꿔 결과를 비교합니다.
2. "협업 경험"과 "같이 일한 경험"이 같은 프로젝트를 찾는지 확인합니다.
3. `chunk_size=100`과 `chunk_size=300`의 검색 결과를 비교합니다.
4. 답변 불가 질문에서 문서에 없는 내용을 지어내지 않는지 확인합니다.
