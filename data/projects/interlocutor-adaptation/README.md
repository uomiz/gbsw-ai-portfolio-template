# Interlocutor Adaptation Dialogue System

## 프로젝트 개요
Incrementally Revealed Persona 환경에서 대화 상대방의 변화에 동적으로 적응하는 대화 생성 프레임워크 연구 및 데이터셋 구축

## 주요 기술
- Natural Language Processing (NLP)
- Persona-based Dialogue
- Dataset Construction

## 기간
2024.03 - 2024.06

## 역할
- 대화 데이터셋 설계 및 구축
- 페르소나 적응 모델 개발
- 성능 평가 지표 설계

## 주요 성과
- 3,000개 이상의 대화 데이터 수집
- 페르소나 변화 감지 정확도 85% 달성
- 학회 논문 제출 준비 중

## 배운 점
대화 시스템에서 동적인 페르소나 변화를 처리하는 방법을 배웠고, 데이터 품질이 모델 성능에 미치는 영향을 깊이 이해하게 되었습니다.
