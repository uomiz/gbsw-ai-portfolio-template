# Dynamic Persona Scheduling Framework

## 프로젝트 개요
대화 문맥에 따라 페르소나의 활성화 필요성을 실시간으로 평가하고 스케줄링하는 모듈형 아키텍처 개발

## 주요 기술
- Large Language Models (LLM)
- Prompt Engineering
- Evaluation Metrics

## 기간
2024.01 - 2024.03

## 역할
- 페르소나 스케줄링 알고리즘 설계
- 프롬프트 엔지니어링을 통한 성능 최적화
- 평가 메트릭 개발

## 주요 성과
- 페르소나 활성화 정확도 92% 달성
- 응답 생성 시간 30% 단축
- 모듈형 아키텍처로 확장성 확보

## 배운 점
LLM의 프롬프트 설계가 성능에 미치는 영향을 실험을 통해 이해했고, 효율적인 평가 지표 설계의 중요성을 배웠습니다.
